Hur man animerar i med Transitions och Animation (@keyframes)

Lägg musen på boxen

och se hur den ändras!

Transitions

Man kan animera med hjälp av transitions:

<div id="box">

#box{
height: 50px;
width: 50px;
background-color: red;
transition: height 2s, background-color 2s;
}

#box:hover{
height: 200px;
width: 50px;
background-color: blue;
}

Använd transition för att visa hur lång tid det kommer att ta att göra ändringen. Därefter skriver du din element och lägger till :hover. Det gör att din transition sker när man håller muspekaren över elementet. I "#box:hover" skriver jag vad det är som kommer att ändras när jag håller muspekaren över box.

@keyframes

Man kan också animera med hjälp av @keyframes:

<div id="box2">

@keyframes animation_with_keyframes{
0% {background-color: black; top: 0px;}
25% {background-color: gray; top: 50px;}
50% {background-color: lightgray; top: 0px;}
75% {background-color: white; left: top: 50px;}
100% {background-color: black; left: top: 0px;}
}

#box2{
height: 50px;
width: 50px;
border: 3px black solid;
background-color: black;
position: relative;
animation-name: animation_with_keyframes;
animation-duration: 5s;
animation-iteration-count: infinite;
}

En animation med @keyframes låter en element ändra från en style till en annan. Man kan ändra så många CSS properties som helst, hur länge som helst.

Jag gjorde att boxen ovanför ska börja svart och sedan ändra gradvis till vit samtidigt som den rör sig upp och ner. Med hjälp av procent berättade jag när den skulle ändras till vad.

Man behöver ge ett namn till animationen och sedan lägger man in animationen med hjälp av egenskapen animation-name. Man behöver också skriva hur långt tid animationen kommer ta med hjälp av animation-duration och även hur många gånger animationen spelas med "animation-iteration-count.
